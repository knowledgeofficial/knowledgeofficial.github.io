Operators:
->An operator performs the operation on data.
  (or) An operator operates on operands.
->An operand is either a constant or a variable.

Types of operators in python:
1.Assignment operator
2.Arithmetic operators
3.Arithmetic Assignment operators
4.Relational operators
5.Logical operators
6.Membership operators
7.Identify operators
8.Conditional operator

1)Assignment Operator(=):
->An assignment operator is used to assign the value to a variable.
Ex:a=100 -> 100 value is assigned to variable a.

2)Arithmetic Operators:
->Arithmetic operators are used to perform the arithmetic operations(mathematical operations)
  on data.
Arithmetic operators in python:
1.+  Addition
2.-  Subtraction
3.*  Multiplication
4./  Division
5.// Floor Division
6.%  Modulation
7.** Power

Note:Division operation gives float result, while floor division operation
     gives integer result.

Ex:a)10//2
     ==>5.0(float result)
    
   b)10//2
     ==>5(integer result)  

3)Arithmetic Assignment Operators:
->Arithmetic Assignment operators will perform two operations.
  a.Arithmetic operation.
  b.Assignment operation.
Arithmetic Assignment operators in python:
1.+=
2.-= 
3.*=
4./=  
5.//= 
6.%=  
7.**= 

Note:a=10,b=20 ==>It will give error.
     a,b=10,2  ==>Right one.

4)Relational (or) Comparision Operators:
->Relational operators are used to check the conditions.
  (or)
->Relational operators are used to compare the values.
1.>
2.<
3.>=
4.<=
5.==
6.!=

5)Logical Operators:
->Logical operators are used to check the multiple conditions at a time.
1.AND
2.OR
3.NOT

Ex:a,b,c,d=10,20,30,40
   a>b and a==d
   False
   b>a and c!=d
   True
   b>a or c==d
   True

6)Membership Operators:
->Membership operators are used to check whether given member(element) is existed or not.
->If given member is existed then it returns True.
->If given member is not existed then it returns False.
->There are two types of membership operators.
1.in
2.not in

Ex:student_names='Ram','Rajesh','Raju','Raina'
   'Ram' in student_names
   True
   'Ramesh' in student_names
   False
   'Ramesh' not in student_names
   True
