Data Types:

-> Data type specifies the type of data.

In python data types classified into 3 types.

1.Fundamental data types.

2. Collection data types.

3. Other data types.

Fundamental Data Types:

Fundamental data types are called as primary data types. These data types are main data types in python, they are..

1)int

2)float

3)string(collection of characters)

4)bool

5)complex

1)int:

In python we have only one numerical data type.. This data type represent a numerical value which doesn't have fractional part.

Ex: eno=123

mobile number = 9848022336

2) float:

Which data belongs to float in python?

A) A number which is having fractional value Which can be +ve or –ve

Ex:10.5

-10.5

3)Str:

which data belongs to string in python ?

A) collection of characters which is enclosed with single codes or double codes or triple codes.

Ex: name='rama'

a=”rama123”

b='”rama@123”'

c='rama@123' (also possible)

4) bool:

bool represent either true or false value , true will be storing as binary value 1 and false as 0 in memory.

5) Complex:

Complex numbers are written a "j" as the imaginary part.

Ex:x=3+5j

     y=5j

     z=-5j

     print(type(x))

     print(type(y))

     print(type(z))

output:(class 'complex')

            (class 'complex')

            (class 'complex')

It is only using at business purpose. (advanced python)

Note:Fundamental data types are enough to work with core python, the remaining "collection data types" and "other data types" are used in advanced python.
