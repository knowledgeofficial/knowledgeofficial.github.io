Q)Write a python program to accept two numbers and perform addition operation and display addition result.

A)a=int(input("Enter first number:"))
  b=int(input("Enter second number:"))
  c=a+b
  print("Addition result is:",c)

output:Enter first number:10
       Enter second number:20
       Addition result is:30

Note:Initially without declaring ant datatype the values or data is stored in string type, for example from the above program if we give a=input("Enter first number") then the value will taken as string as "10" then addition operation for string values would be "10"+"5"=105, usually performing addition operation between two operands it will perform concatination operation, so for that we are stpring the values as interger data as a=int(input("Enter first number:")).
If you observe here we are using type casting from string to integer data.
