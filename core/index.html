6) python is a dynamic typed programming language.

Means programmer doesn't required to declare data type based on the value you have given, automatically type will decide.

7) Python is a interpreter based programming language.

programming languages are two types

a)compiler based.

b)interpreter based

In. compiler based programming languages, the program will first compile, once compilation is completed, then only it will get execute.

Ex:c,c++,java,c#.net and so on.

In interpreter based programming, here interpreter means execution engine, it will compile and execute the code line by line.

8) Python is an embedded programming language.

That means...

->we can integrate Python code into java code as well as dot net code.

->To integrate python code into java code we have Jython.

Note:

If we want to integrate python code into java code, we hawe to install Jython software, on that software we should open. Java application and we can write Python code which will convert as java code by jython.. Now we can run this application on java frameworks.

-> How to integrate python code into dot net code?

For this, we have IRON Python software.
