LOOPS:

->Using loop[s we can execute a single statement or a block of statements multiple times.
->Whenever we want to execute a single statement or multiple statements more than one time,we will go with loops.

Types of Loops:

1)for loop:

syntax:for (variablename) in range((starting range),(ending range+1)):
        #logic

Ex-1:Write a program to print 1 to 5 numbers by using for loop.
A)for i in range(1,6):
  print(i) 

output:
1
2
3
4
5

Note:Above code will perform first "initilization" and "comparing" and then "increment".

Ex-2:for i in range(1,6):
      print(i,end=" ")
output:1 2 3 4 5 

Ex-3:Write a python program to print numbers from 5 to 1.

A)for i in range(5,0,-1):
  print(i,end=" ")
output:5 4 3 2 1

Note:By default range will increment by 1,so we want to decrement the values for that we are declaring third value as "-1",which is used for decrementing the values.

range()

It will support three overloaded methods.

1)range(value)

This method will be used in collections.

2)range((starting range),(ending range+1))

This method will do
->initilizing value.
->comparing value with ending range.
->after the above execution,it will go for iteration.
->after iteration,it will increment the value+1.

3)range((starting range),(ending range+1),(increment or decrement))

This method will do
->initilizing value.
->comparing value with ending range.
->after the above execution,it will go for iteration.
->after iteration,it will go for increment or decrement.

Transfer Statements:

->Using transfer statements we can transfer the program control from one statement to another statement.
->Whenever we want to transfer the program control from one place to another place.
->In python we have three types of transfer statements.
  1.break
  2.continue
  3.pass

break:

->break will transfer the program control out of the loop.

Q)Write a python code to print 1 to 10 numbers,but output should stop at number 6.

A)for i in range(1,11):
     print(i,end=" ")
     if i==6:
      break

output:1 2 3 4 5 6

continue:

->continue will skip the current iteration of the loop.

Q)Write a program to print 1 to 10 numbers by skipping number 5 in the list.

A)for i in range(1,11):
   if i==5:
   continue
   print(i,end=" ")

output:1 2 3 4 6 7 8 9 10

pass:

->Using pass we can pass control from one block to another block,this pass we will use specially at the time of functions.

Ex:for i in range(1,6):
      if i==3:
       pass
      print(i,end=" ")
output:1 2 3 4 5
