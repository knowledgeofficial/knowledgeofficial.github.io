 Type casting for fundamental data types:

what is type casting? 
A)Type casting is a process of converting one data type to another data type. 
-> Python will support following type casting methods, 
1)int() 
2) float()
3) Str()
4) bool()
5) complex() 

1)Int(): 
This method converts the given value to int type.
Ex(a):a=10.5
      int(a) 
output:10
it is converting float to int.

Ex(b): a=”10”
       int(a)  
output:10
It is converting string to int.

Ex(c):a=True
      int(a)
output:1
It is converting bool to int.

Ex(d):a=’rama’
      int(a)
output:error

2)float():
This method will convert the given value into float type.

Ex(a): a=10 
       float(a) 
output:
10.0
 It is converting from int to float.

Ex(b):a="10.5"
      float(a) 
output:
10.5 
It is converting from string to float.

Ex(c):float (true) 
output:1.0
 This method will convert from bool to float.

3)str():
 This method will convert from any type to string.

Ex(a):str(10)
output:'10'
It is converting from int to string.

Ex(b):str(10.5)
output:'10.5'
It is converting from float to string.

Ex(c):str(True)
output:'True'
It is converting from bool to string.

4)bool():
This method will convert the given type into bool type.

Ex(a):bool(10)
output:True
It is converting from int to bool.

Ex(b):bool(10.5)
output:True
It is converting from float to bool.

Ex(c):bool(-10)
output:True
It is converting from negative integer value to bool.

Ex(d):bool("rama")
output:True
It is converting from string to bool.
